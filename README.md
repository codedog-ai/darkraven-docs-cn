# 欢迎来到 Darkraven

Darkraven 是面向大语言模型（LLM）应用的可观测平台。它的目标是帮助用户实现可观测性，并提供自动化的生成内容评估能力，以满足用户在 LLM 输出效果、质量、风控等方面的分析和监控需求。

## 功能

* **LLM 链路**
* **LLM 生成评估**
* **指标**
* **仪表盘**
* **告警**
